# 个人小天地

一个专属于你和TA的私密网站，用于存储和分享文章、随笔、图片和视频。

## 功能特点

- 文章管理：支持上传和查看 Markdown/TXT 格式的文章
- 随笔记录：随时记录生活中的点滴感悟
- 图片存储：支持 JPG、PNG 等常见格式，以及 RAW 格式（CR2、NEF、ARW、DNG 等）
- 视频播放：支持常见视频格式的在线播放
- 本地存储：所有数据保存在浏览器本地，保护隐私
- 响应式设计：完美适配手机和电脑

## 使用方法

1. 访问网站
2. 点击相应板块的"添加"按钮上传文件
3. 图片支持按格式筛选（全部/JPG/RAW）
4. 点击文件可查看详情或下载原图

## 技术栈

- HTML5
- CSS3
- Vanilla JavaScript
- LocalStorage 本地存储

## 部署

本网站使用 GitHub Pages 自动部署。
